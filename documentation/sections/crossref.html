BaseMeasurec back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#BaseMeasure

Base measure is a quality measure for which, in the process of evaluation, the results are obtained in an individual test and are related to direct outputs of the evaluation.

QualityMeasure c
is disjoint with
QualityIndicator c, DerivedMeasure c

OrdinalScaleItemc back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#OrdinalScaleItem

Ordinal scale item is an item that belongs to a particular ordinal scale. Ordinal scale item is described with a label (e.g., "Excellent"/"Good"/...) and an order on the scale (e.g., 1/2/...).

is in domain of
hasLabel dp, hasOrder dp
is in range of
hasOrdinalScaleItem op
is disjoint with
SubjectCategory c, QualityCharacteristic c, QualityMeasure c, RankingFunction c, Measurement_scale c, Unit_of_measure c

QualityCharacteristicc back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#QualityCharacteristic

Quality characteristic is a characteristic of a product, service or action that can be measured in an evaluation using quality measures.

is in domain of
hasSubCharacteristic op, isCharacteristicOf op, isMeasuredWith op
is in range of
hasSubCharacteristic op, measuresCharacteristic op
is disjoint with
SubjectCategory c, OrdinalScaleItem c, QualityMeasure c, RankingFunction c, Measurement_scale c, Unit_of_measure c

QualityIndicatorc back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#QualityIndicator

Quality indicator is a quality measure that is obtained by combining base and/or derived measures. Unlike base and derived measures, quality indicators are related to a set of tests in the evaluation.

has super-classes
QualityMeasure c
is disjoint with
BaseMeasure c, DerivedMeasure c

QualityMeasurec back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#QualityMeasure

Quality measure is any measure that captures information about a specific quality characteristic related to a specific product, service or action.

has sub-classes
BaseMeasure c, DerivedMeasure c, QualityIndicator c
is in domain of
collaboratesWith op, damages op, dependsOn op, hasScale op, hasUnitOfMeasurement op, isUsedToObtain op, measuresCharacteristic op
is in range of
collaboratesWith op, damages op, dependsOn op, isMeasuredWith op, isUsedToObtain op
is disjoint with
SubjectCategory c, OrdinalScaleItem c, QualityCharacteristic c, RankingFunction c, Measurement_scale c, Unit_of_measure c

RankingFunctionc back to ToC or Class ToC

IRI: http://purl.org/net/QualityModel#RankingFunction

Ranking function is a function which denotes how two values of a measurement scale can be ranked, i.e., if higher or lower value is better.

is equivalent to
{HigherBest, LowerBest}
is in range of
hasRankingFunction op
is disjoint with
