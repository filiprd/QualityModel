QualityIndicatorc back to ToC or Class ToC

IRI: http://purl.org/net/QualityModelOntology#QualityIndicator

Quality indicator is a quality measure that is obtained by combining base and/or derived measures. Unlike base and derived measures, quality indicators are related to a set of tests in the evaluation.

has super-classes
QualityMeasure c
is disjoint with
BaseMeasure c, DerivedMeasure c
